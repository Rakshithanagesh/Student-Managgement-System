# 🎓 Student Management System

## 📌 Description

A full-stack Student Management System developed using Django and React to efficiently manage and organize student records. The application enables users to perform CRUD operations (Create, Read, Update, Delete) through a responsive and user-friendly interface.

---

## 🚀 Features

* Add, update, and delete student records
* View student details in a structured format
* Responsive frontend built with React
* REST API integration with Django backend
* Efficient data handling using SQLite database

---

## 🛠 Tech Stack

* **Frontend:** React (HTML, CSS, JavaScript)
* **Backend:** Django (Python, Django REST Framework)
* **Database:** SQLite

---

## 📂 Project Structure

```
student-management-system/
│── backend/      # Django backend
│── frontend/     # React frontend
```

---

## ⚙️ Installation & Setup

### 🔹 Backend (Django)

```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```

### 🔹 Frontend (React)

```bash
cd frontend
npm install
npm start
```

---

## 🌐 Usage

* Open browser and go to:
  http://127.0.0.1:8000/
* Use the interface to manage student records
